🌐 Chapter 11: Introduction to Web APIs and JSON

🤔 Why Learn APIs?

“APIs are like waiters in a restaurant — they take your order to the kitchen and bring back the food!”

  • Get live weather, news, sports scores 📰
  • Access data from platforms like YouTube, Twitter, or Google 🌍
  • Control smart devices, chatbots, and more 💬

📦 What is an API?

API stands for Application Programming Interface. It lets two programs talk to each other.

🍔 Real-Life Analogy:

  • You 🧑: “I want a pizza!”
  • Waiter 🧑‍🍳 (API): Takes your order to the kitchen.
  • Kitchen 🔥 (Server): Makes the pizza (data).
  • Waiter brings it back to your table.

Same way:
🧑 (Your Python app) → 🌐 API → 💾 Server → 🌐 API → 🧑 (Your app gets data!)

🔢 What is JSON?

JSON = JavaScript Object Notation

It's the language of data used by most APIs. It's just a text format to send structured data.

👇 Example JSON:

{
  "name": "Aman",
  "age": 20,
  "subjects": ["Math", "Python", "AI"]
}

Looks like a Python dictionary, right? Exactly!

🛠️ How to Work with APIs in Python

You’ll use a module called requests:

pip install requests

🔗 Example: Fetching Data from a Free API

import requests
response = requests.get("https://api.agify.io/?name=aman")
data = response.json()
print(data)
print("Predicted Age:", data["age"])

Output:
{'name': 'aman', 'age': 25, 'count': 100}
Predicted Age: 25

🧰 Useful API Terms

Term Meaning
URL Web address of the API
GET Request to fetch data
POST Request to send data
JSON Format used for request/response

🤪 Funny Analogy

JSON is like your friend who texts you everything in perfect grammar and bullet points 😎

API is the delivery boy between your app and a powerful data warehouse 📦

🌦 Real-World Example: Get Weather Info

import requests
city = "London"
url = f"http://api.weatherapi.com/v1/current.json?key=your_api_key&q={city}"
response = requests.get(url)
data = response.json()
print("Temperature:", data['current']['temp_c'], "°C")

(Note: You need an API key 🔑 — sign up free on weatherapi.com)
