🔤 What is a String?

A string is a sequence of characters. In Python, strings are immutable (cannot be changed after creation).

Analogy: Think of a string as a row of lockers — you can read what's inside, but you can't swap the doors! 
name = "Aman"
quote = "Python is fun!"

🧪 String Indexing & Slicing

Access parts of a string using indexing and slicing:

text = "Python"
print(text[0])      # P   (first character)
print(text[-1])     # n  (last character)
print(text[1:4])    # yth (2nd to 4th character)
print(text[::-1])   # nohtyP (reversed)
💡 Indexing Recap:
Positive: 0, 1, 2... | Negative: -1, -2, ...
CharPython
Pos012345
Neg-6-5-4-3-2-1
🔧 Slicing Syntax: string[start:end:step]
ParameterDescription
startindex to begin from (inclusive)
endindex to stop before (exclusive)
stephow many steps to skip (default 1)

🔧 Common String Methods – Explained with Examples

1. .lower() – Convert to lowercase

✅ Converts all characters in a string to lowercase.

text = "PYTHON"
    print(text.lower())  # Output: python

2. .upper() – Convert to uppercase

✅ Converts all characters to uppercase.

greeting = "hello"
    print(greeting.upper())  # Output: HELLO

3. .strip() – Remove leading & trailing spaces

✅ Removes unwanted spaces from both ends.

name = "   Aman   "
    print(name.strip())  # Output: 'Aman'

4. .replace() – Replace characters/substrings

✅ Replaces parts of the string with something else.

fruit = "apple"
    print(fruit.replace("p", "b"))  # Output: abble

5. .find() – Find the index of a substring

✅ Returns the first index of a substring, or -1 if not found.

msg = "hello"
    print(msg.find("e"))  # Output: 1

6. .split() – Split string into a list

✅ Splits the string by the given delimiter.

items = "a,b,c"
    print(items.split(","))  # Output: ['a', 'b', 'c']

7. .join() – Join list into a string

✅ Joins elements of a list into a single string using a separator.

letters = ['a', 'b', 'c']
    print("-".join(letters))  # Output: a-b-c

8. .isdigit() – Check if string is numeric

✅ Returns True if all characters in the string are digits.

num = "12345"
    print(num.isdigit())  # Output: True
    
    word = "abc123"
    print(word.isdigit())  # Output: False

🔁 Bonus Practical Example:

email = "   Hello@Python.com   "
    cleaned = email.strip().lower().replace("@", "[at]")
    print(cleaned)
    # Output: hello[at]python.com

📘 Part 2: File I/O (Input/Output)

File handling lets your Python program read from or write to external files like .txt, .csv, .json, etc.

📝 1. Opening a File

Use the open() function to access a file.

file = open("notes.txt", "r")  # Open in read mode
  content = file.read()          # Read content
  print(content)
  file.close()                   # Close file

"notes.txt" is the file name.

"r" means "read mode".

🖊️ Writing & Appending to a File

file = open("notes.txt", "w")  # Open in write mode
file.write("Learning Python is fun!\n")
file.close()
➕ Append mode adds to the end of the file without deleting existing data. 
file = open("notes.txt", "a")  # Open in append mode
file.write("Let's master file handling.\n")
file.close()

📖 Reading Line-by-Line

file = open("notes.txt", "r")
for line in file:
    print(line.strip())
file.close()
📚 Tip: .strip() removes \n from each line when printed.

✅ Using with (Best Practice)

with open("notes.txt", "r") as f:
    content = f.read()
    print(content)
👍 Why? with auto-closes the file, even if errors occur!

📂 Student Data Example

with open("student.txt", "w") as f:
    f.write("Name: Aman\n")
    f.write("Course: Python\n")
    f.write("Marks: 88\n")
📖 To read it: 
with open("student.txt", "r") as f:
    print(f.read())

🔄 User Input to File

data = input("Enter something to save: ")
with open("mydata.txt", "w") as file:
    file.write(data)
