‚úÖ 1. Introduction to Functions

üîπ What is a Function?

A function is a reusable block of code that performs a specific task.

üí° Analogy: Think of a function as a coffee machine: you give it ingredients (inputs), press a button (call the function), and get coffee (output)!

üîπ Why Use Functions?

  • To organize code.
  • To avoid repetition.
  • To break problems into smaller pieces.

üß† Syntax of a Function in Python:

    # block of code
    return result

‚úÖ 3. The return Statement in Python Functions

üîπ What is return?

The return statement is used to send a result back to the place where the function is called.
It ends the function execution immediately.

‚úÖ Example 1: Simple Return

def square(num):
    return num * num

result = square(4)
print("Square is:", result)

üü© Output: Square is: 16

‚úÖ Example 2: Return Multiple Values

def get_values():
    return 10, 20, 30

a, b, c = get_values()
print(a, b, c)

üü© Output: 10 20 30

‚úÖ Example 3: Return vs Print

def show():
    print("Hello")

def give():
    return "Hello"

x = show()     # prints Hello, but x is None
y = give()     # does not print but y gets the value "Hello"

print("x:", x)  # Output: x: None
print("y:", y)  # Output: y: Hello

‚úÖ 5. Lambda (Anonymous) Functions

üîπ What is a Lambda Function?

A lambda function is a small anonymous function ‚Äî meaning it doesn‚Äôt have a name like regular functions created with def.

üìå Syntax:

lambda arguments: expression

‚úÖ It can take any number of arguments, but only one expression.

‚úÖ Example 1: Simple Lambda Function

square = lambda x: x * x
print(square(5))  # Output: 25

‚úÖ Example 2: Lambda with Multiple Arguments

add = lambda a, b: a + b
print(add(3, 4))  # Output: 7

‚úÖ Example 3: Using Lambda with map()

nums = [1, 2, 3, 4]
squared = list(map(lambda x: x ** 2, nums))
print(squared)  # Output: [1, 4, 9, 16]

‚úÖ Example 4: Using Lambda with filter()

nums = [5, 8, 12, 3, 7]
even = list(filter(lambda x: x % 2 == 0, nums))
print(even)  # Output: [8, 12]

‚úÖ 6. Modules in Python

üîπ What is a Module?

A module is a file containing Python code (functions, variables, classes) that you can import and reuse in other programs.

üìÅ Think of a module as a toolbox ‚Äî you import it when you need its tools (code).

üîπ Why Use Modules?

  • Reusability: Write once, use anywhere.
  • Organization: Keep code clean and separate.
  • Standard Library: Use built-in powerful features like math,random,datetime.

‚úÖ Types of Modules

‚úÖ Built-in Modules in Python

These modules come pre-installed with Python. You just import them and start using ‚Äî no installation needed.

üì¶ 1. math Module

Used for mathematical operations.

import math

print(math.sqrt(25))         # Square root: 5.0
print(math.ceil(4.2))        # Ceiling value: 5
print(math.floor(4.9))       # Floor value: 4
print(math.pow(2, 3))        # 2^3 = 8.0
print(math.pi)               # 3.141592...

üé≤ 2. random Module

Used to generate random numbers, choose random items, etc.

import random

print(random.randint(1, 10))         # Random integer between 1 and 10
print(random.choice(['apple', 'banana', 'cherry']))  # Random choice
print(random.random())               # Random float between 0 and 1

üíª 3. os Module

Provides functions to interact with the Operating System.

import os

print(os.getcwd())           # Current working directory
print(os.name)               # OS name (e.g., 'nt' for Windows, 'posix' for Unix)
os.mkdir("new_folder")       # Create new folder
os.remove("file.txt")        # Delete a file

‚è∞ 4. datetime Module

Used for working with dates and times.

import datetime

now = datetime.datetime.now()
print("Current Date and Time:", now)

today = datetime.date.today()
print("Today's Date:", today)

delta = datetime.timedelta(days=5)
future = today + delta
print("5 days from today:", future)

‚úÖ User-Defined Modules in Python

üîπ What is a User-Defined Module?

A user-defined module is simply a Python file (.py) that contains functions, variables, or classes ‚Äî created by you ‚Äî and can be imported into other Python files.

üì¶ Imagine making your own toolbox that you can reuse in other scripts.

‚úÖ Step-by-Step Example:

üî∏ Step 1: Create the Module File

üìù mytools.py

# This is your user-defined module

def greet(name):
    return f"Hello, {name}!"

def add(a, b):
    return a + b

pi = 3.14159

üî∏ Step 2: Use it in another file

üìù main.py

import mytools

print(mytools.greet("Aman"))      # Output: Hello, Aman!
print(mytools.add(10, 5))         # Output: 15
print(mytools.pi)                 # Output: 3.14159

‚úÖ You can also import specific functions:

from mytools import greet, pi

print(greet("Raj"))
print(pi)

üîπ How It Works:

  • Python searches for mytools.py in the current folder or sys.path.
  • It loads the module once and caches it.
  • A .pyc (compiled) version may also be created for faster loading.

üß† Tips:

  • File name becomes module name: mymodule.py ‚Üí import mymodule
  • Avoid spaces or special characters in filenames.
  • Good for organizing large programs into smaller pieces.
