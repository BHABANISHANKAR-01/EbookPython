🖼️ What is matplotlib?

matplotlib is a powerful plotting library in Python that turns raw data into beautiful charts.

It’s like the artist in your Python toolkit 🎨

✍️ Basic Usage

import matplotlib.pyplot as plt
x = [1, 2, 3, 4]
y = [10, 20, 25, 30]
plt.plot(x, y)
plt.title("Basic Line Chart")
plt.xlabel("X Axis")
plt.ylabel("Y Axis")
plt.show()

Output: 🟢 A simple line chart with labeled axes and title.

🧱 Types of Charts You Can Make

Chart Type Function Use Case
Line Chart plot() Track change over time
Bar Chart bar() Compare categories
Pie Chart pie() Show parts of a whole
Histogram hist() Frequency of values
Scatter Plot scatter() Show relationships

📊 Bar Chart Example

import matplotlib.pyplot as plt
students = ["Aman", "Elon", "John"]
marks = [85, 90, 75]
plt.bar(students, marks, color='skyblue')
plt.title("Student Marks")
plt.xlabel("Names")
plt.ylabel("Marks")
plt.show()

🥧 Pie Chart Example

subjects = ["Math", "Science", "English"]
time_spent = [5, 3, 2]
plt.pie(time_spent, labels=subjects, autopct='%1.1f%%')
plt.title("Time Spent per Subject")
plt.show()

📉 Scatter Plot Example

x = [1, 2, 3, 4, 5]
y = [2, 4, 1, 8, 7]
plt.scatter(x, y, color='red')
plt.title("Scatter Plot")
plt.xlabel("X values")
plt.ylabel("Y values")
plt.show()
