💰 Project Name: Compound Interest Calculator

✅ Project Overview:

This program will calculate Compound Interest using the standard formula:

A = P(1 + r/n)^(nt)

Where:

  • A = Final amount
  • P = Principal amount
  • r = Annual interest rate (in decimal)
  • n = Number of times interest is compounded per year
  • t = Time (in years)

Compound Interest = A - P

🔁 Python Code:

def calculate_compound_interest(principal, rate, time, n):
    # Convert percentage rate to decimal
    r = rate / 100
    amount = principal * (1 + r / n) ** (n * time)
    ci = amount - principal
    return amount, ci

while True:
    print("\nCompound Interest Calculator")
    print("1. Calculate Compound Interest")
    print("2. Exit")

    choice = input("Enter your choice (1-2): ")

    if choice == '2':
        print("Thank you for using the calculator. Goodbye!")
        break

    principal = float(input("Enter the principal amount (P): ₹"))
    rate = float(input("Enter the annual interest rate (in %): "))
    time = float(input("Enter the time in years (T): "))
    n = int(input("Enter number of times interest is compounded per year (n): "))

    total_amount, compound_interest = calculate_compound_interest(principal, rate, time, n)

    print(f"\nTotal Amount (A) after {time} years = ₹{total_amount:.2f}")
    print(f"Compound Interest earned = ₹{compound_interest:.2f}")

🔍 Code Explanation:

  • calculate_compound_interest(...): Performs the math using the compound interest formula.

import time

def countdown_timer(seconds):
    while seconds:
        mins, secs = divmod(seconds, 60)
        timer = f'{mins:02d}:{secs:02d}'
        print(f"\rTime Left: {timer}", end="")
        time.sleep(1)
        seconds -= 1
    print("\rTime's up!           ")

while True:
    print("\n⏳ Countdown Timer")
    print("1. Start Timer")
    print("2. Exit")

    choice = input("Enter your choice (1-2): ")

    if choice == '2':
        print("Goodbye!")
        break

    total_seconds = int(input("Enter time in seconds: "))
    countdown_timer(total_seconds)

🔍 Code Explanation:

  • divmod(seconds, 60): Splits seconds into minutes and seconds
  • \r: Returns cursor to the beginning of the line (for live countdown)
  • end="": Prevents printing on a new line each time
  • time.sleep(1): Waits for 1 second between countdown steps
  • After countdown, it prints "Time's up!"

✊✋✌️ Project Name: Rock, Paper, Scissors Game

✅ Project Overview:

In this game:

  • The player chooses Rock, Paper, or Scissors.
  • The computer randomly chooses one too.
  • The winner is decided based on the classic rules:
    • Rock beats Scissors
    • Scissors beats Paper
    • Paper beats Rock
  • It keeps track of the score and allows multiple rounds.

🧠 Python Concepts Used:

  • random.choice() for computer move
  • input() for user move
  • if/elif/else for game logic
  • Score tracking with variables

🎮 Full Python Code:

import random

choices = ["rock", "paper", "scissors"]

def get_winner(player, computer):
    if player == computer:
        return "draw"
    elif (player == "rock" and computer == "scissors") or \
         (player == "paper" and computer == "rock") or \
         (player == "scissors" and computer == "paper"):
        return "player"
    else:
        return "computer"

player_score = 0
computer_score = 0

while True:
    print("\n🎮 Rock, Paper, Scissors Game")
    print("Choose one: rock, paper, scissors")
    player_choice = input("Your choice: ").strip().lower()

    if player_choice not in choices:
        print("❌ Invalid choice. Try again.")
        continue

    computer_choice = random.choice(choices)
    print(f"🤖 Computer chose: {computer_choice}")

    result = get_winner(player_choice, computer_choice)

    if result == "draw":
        print("🔁 It's a draw!")
    elif result == "player":
        print("✅ You win this round!")
        player_score += 1
    else:
        print("❌ You lose this round.")
        computer_score += 1

    print(f"Score → You: {player_score} | Computer: {computer_score}")

    play_again = input("🔁 Play again? (yes/no): ").strip().lower()
    if play_again != "yes":
        print("👋 Thanks for playing!")
        break

🎯 Project Name: Python Slot Machine Game

This game simulates a slot machine where users place a bet, the machine spins symbols, and payouts are based on matches.

✅ Project Overview:

  • Symbols: 🍒, 🍋, 🍊, 💎, 7️⃣, 🍉
  • User places a bet
  • Machine spins 3 symbols
  • Match symbols to win
  • Tracks and updates balance

🧠 Concepts Used:

  • random.choice() to simulate spinning
  • Functions and game loop
  • Balance management
  • Symbols and payouts logic

🛠️ Full Python Code:

import random

# Available symbols
symbols = ["🍒", "🍋", "🍊", "💎", "7️⃣", "🍉"]

# Function to simulate spinning the slot
def spin():
    return [random.choice(symbols) for _ in range(3)]

# Function to calculate payout
def calculate_payout(result, bet):
    if result[0] == result[1] == result[2]:
        print("🎉 Jackpot! All 3 symbols match!")
        return bet * 10
    elif result[0] == result[1] or result[1] == result[2] or result[0] == result[2]:
        print("✨ Two symbols match! Small win!")
        return bet * 2
    else:
        print("😞 No match. You lose.")
        return -bet

# Main game loop
def play_slot_machine():
    balance = 1000  # Starting balance

    print("🎰 Welcome to the Python Slot Machine!")
    print(f"💰 You start with ₹{balance}")

    while balance > 0:
        print(f"\nYour current balance: ₹{balance}")
        try:
            bet = int(input("Place your bet (or 0 to quit): ₹"))
        except ValueError:
            print("❌ Invalid input. Enter a number.")
            continue

        if bet == 0:
            print("👋 Thanks for playing!")
            break
        elif bet < 0:
            print("❌ Bet can't be negative.")
            continue
        elif bet > balance:
            print("❌ You can't bet more than your balance.")
            continue

        result = spin()
        print("Spinning...")
        print(" | ".join(result))

        winnings = calculate_payout(result, bet)
        balance += winnings

    if balance <= 0:
        print("💸 You're out of money! Game over.")

# Start the game
play_slot_machine()

🧠 How Winning Works:

Match Type Reward
3 of a kind bet × 10
2 of a kind bet × 2
No match Lose your bet
