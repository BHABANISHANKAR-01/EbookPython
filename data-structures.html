🧃 1. Python List – A Basket of Items 🧺

🔍 What is a List?

A list in Python is an ordered, mutable (changeable) collection of items, enclosed in square brackets [].

✅ A list can hold:

  • Strings: ["apple", "banana"]
  • Numbers: [1, 2, 3]
  • Mixed types: ["Aman", 25, True]
  • Even other lists: [[1, 2], ["x", "y"]]

🛍️ Funny Analogy

A list is like a shopping basket – you can:

  • Add items 🍎
  • Remove items 🥭
  • Sort them 📦
  • Count how many are inside 🧮
fruits = ["apple", "banana", "mango"]
print(fruits[1])  # Output: banana (Lists are 0-indexed!)

⚙️ Useful List Methods

🧠 Accessing List Items

fruits = ["apple", "banana", "cherry"]
print(fruits[0])    # apple
print(fruits[-1])   # cherry

🔄 Slicing a List

fruits = ["apple", "banana", "mango", "grape", "orange"]
print(fruits[1:4])    # ['banana', 'mango', 'grape']
print(fruits[:3])     # ['apple', 'banana', 'mango']
print(fruits[-3:])    # ['mango', 'grape', 'orange']

🪑 2. Tuple – The Unchangeable Chair Set

🔍 What is a Tuple?

A tuple is an ordered, immutable (unchangeable) collection of items, enclosed in parentheses ().

🧠 Funny Analogy

A tuple is like your school timetable 📅 or a chair set 🪑:
Once made, you can’t change or remove any item. You can sit on it, count it, and admire it — but you can't modify it! 

colors = ("red", "green", "blue")
print(colors[0])  # Output: red

✅ Why Use Tuples?

ReasonExplanation
✅ Faster than listsTuples take less memory and are faster
✅ ImmutableProtect data from being changed
✅ Used as dictionary keysOnly immutable types can be keys
✅ Safer data representationGood for read-only or fixed data

🎯 Tuple Syntax:

# Normal tuple
days = ("Mon", "Tue", "Wed")

# Single-element tuple (must include comma!)
one = ("hello",)
print(type(one))  # Output: <class 'tuple'>

🔁 3. Set – A Unique Collection 🌟

🔍 What is a Set in Python?

A Set is an unordered, mutable, and unindexed collection that stores only unique elements.

✅ Duplicate values are automatically removed.

🧠 Funny Analogy:

A set is like a VIP list at a party 🎉 — each guest (element) can appear only once. No duplicates allowed!

🧪 Example:

numbers = {1, 2, 2, 3, 4}
print(numbers)  # Output: {1, 2, 3, 4}

📌 Key Characteristics of Sets

PropertyDescription
✅ No duplicatesAll elements are unique
🔄 UnorderedNo indexing or order guaranteed
🚀 Fast lookupFaster than lists for checking existence
🛠️ MutableYou can add/remove items

🧠 Real Example:

emails = {"a@example.com", "b@example.com", "a@example.com"}
print(emails)  # Output: {'a@example.com', 'b@example.com'}

✅ Common Set Methods

MethodDescriptionExample
add(x)Adds items.add(5)
remove(x)Removes item; error if not founds.remove(3)
discard(x)Removes item; no error if not founds.discard(3)
clear()Removes all itemss.clear()
pop()Removes random items.pop()
union(set2)Combines sets (no duplicates)s1.union(s2)
intersection(set2)Returns common elementss1.intersection(s2)
difference(set2)Items only in first sets1.difference(s2)
issubset(set2)Checks if set1 ⊆ set2s1.issubset(s2)
issuperset(set2)Checks if set1 ⊇ set2s1.issuperset(s2)

🧪 Code Example

set1 = {1, 2, 3}
set2 = {3, 4, 5}

print(set1.union(set2))         # {1, 2, 3, 4, 5}
print(set1.intersection(set2))  # {3}
print(set1.difference(set2))    # {1, 2}

🤖 4. Dictionary – Your Phone Contacts 📱

🔍 What is a Dictionary in Python?
A dictionary is an unordered, mutable collection of key-value pairs, where each key is unique and used to access its corresponding value.

🧠 Funny Analogy:
A dictionary is like your phonebook 📱: You search a name (key) and get the number or info (value). No duplicate names allowed!

✅ Example:

student = {
  "name": "Aman",
  "age": 21,
  "marks": 88
}

print(student["name"])  # Output: Aman

🧠 Real-World Example – Nested Dictionary

students = {
  "Aman": {"age": 21, "course": "Python"},
  "Ayush": {"age": 20, "course": "Java"}
}

print(students["Ayush"]["course"])  # Output: Java

📌 Key Characteristics

  • 🔑 Keys are unique – No duplicate keys allowed
  • 🔁 Mutable – You can change, add, delete items
  • ❌ No index – You access values using keys, not positions
  • 🧠 Powerful – Supports nested structures (dict in dict)

🎯 Useful Dictionary Methods

MethodDescriptionExample
dict[key]Access value by keystudent["name"]
dict[key] = valueAdd or update valuestudent["course"] = "Python"
pop(key)Remove key-value pairstudent.pop("age")
keys()Returns list of keysstudent.keys()
values()Returns list of valuesstudent.values()
items()Returns key-value pairsstudent.items()
get(key, default)Safer way to get valuestudent.get("marks", 0)
clear()Remove everythingstudent.clear()
update(dict2)Add items from another dictionarystudent.update({"age": 22})

🧪 Example Code

student = {
  "name": "Aman",
  "age": 21,
  "course": "Python"
}

print(student["name"])               # Aman
student["marks"] = 95                # Add new item
student.pop("age")                   # Remove item
print(student.keys())                # dict_keys(['name', 'course', 'marks'])
print(student.values())              # dict_values(['Aman', 'Python', 95])

💡 Bonus Tip: Dictionary Comprehension

squares = {x: x**2 for x in range(1, 4)}
print(squares)  # {1: 1, 2: 4, 3: 9}

⚠️ Common Mistakes

  • Using unhashable types as keys: ❌ TypeError (e.g., lists as keys)
  • Accessing a missing key without get(): ❌ KeyError
  • Forgetting quotes on string keys: ❌ NameError

✅ Best Practice:

print(student.get("grade", "Not Available"))

🎓 Mini Project: Student Management System

✅ Objective: Create a program to manage students’ info using:

  • 🛍️ List to store multiple students
  • 🪑 Tuple to store fixed data like available courses
  • 🔁 Set to store unique clubs or hobbies
  • 📱 Dictionary to store student details

✅ Project Features:

  • Add new student
  • Show all students
  • Search student by name
  • List all enrolled courses
  • List all unique clubs

💻 Full Python Code:


  # Fixed list of available courses (Tuple)
  available_courses = ("Python", "Java", "C++", "HTML", "DSA")
  
  # Student records (List of dictionaries)
  students = []
  
  # Set to store unique clubs
  all_clubs = set()
  
  # Main loop
  while True:
      print("\n🎓 Student Management System")
      print("1. Add Student")
      print("2. Show All Students")
      print("3. Search by Name")
      print("4. Show All Courses")
      print("5. Show All Unique Clubs")
      print("6. Exit")
  
      choice = input("Enter your choice: ")
  
      if choice == '1':
          name = input("Enter student name: ")
          age = int(input("Enter age: "))
          course = input(f"Enter course ({', '.join(available_courses)}): ")
  
          if course not in available_courses:
              print("❌ Course not available.")
              continue
  
          club = input("Enter club name: ")
  
          student = {
              "name": name,
              "age": age,
              "course": course,
              "club": club
          }
  
          students.append(student)
          all_clubs.add(club)
          print("✅ Student added successfully!")
  
      elif choice == '2':
          if not students:
              print("No students added yet.")
          else:
              for s in students:
                  print(f"👤 {s['name']}, Age: {s['age']}, Course: {s['course']}, Club: {s['club']}")
  
      elif choice == '3':
          search = input("Enter name to search: ")
          found = False
          for s in students:
              if s['name'].lower() == search.lower():
                  print(f"✅ Found: {s}")
                  found = True
          if not found:
              print("❌ Student not found.")
  
      elif choice == '4':
          print("🎓 Courses Offered:")
          for c in available_courses:
              print(f"- {c}")
  
      elif choice == '5':
          print("🏆 Unique Clubs:")
          for club in all_clubs:
              print(f"- {club}")
  
      elif choice == '6':
          print("👋 Exiting program. Goodbye!")
          break
  
      else:
          print("❌ Invalid choice. Please try again.")

  • Load data from the file when the program starts

  • json module

    import json

    # Load data if file exists
    if os.path.exists(filename):
        with open(filename, "r") as file:
            students = json.load(file)
            all_clubs = set([s["club"] for s in students])
    
    # Main loop
    while True:
        print("\n🎓 Student Management System")
        print("1. Add Student")
        print("2. Show All Students")
        print("3. Search by Name")

            name = input("Enter student name: ")

            course = input(f"Enter course ({', '.join(available_courses)}): ")
    
            if course not in available_courses:
                print("❌ Course not available.")
                continue
    
            club = input("Enter club name: ")
    
            student = {
                "name": name,
                "age": age,
                "course": course,
                "club": club
            }
    
            students.append(student)
            all_clubs.add(club)
    
            # Save to file
            with open(filename, "w") as file:
                json.dump(students, file, indent=4)
    
            print("✅ Student added and saved!")
    
        elif choice == '2':
            if not students:
                print("No students yet.")
            else:
                for s in students:
                    print(f"👤 {s['name']}, Age: {s['age']}, Course: {s['course']}, Club: {s['club']}")
    
        elif choice == '3':
            search = input("Enter name to search: ")
            found = False
            for s in students:
                if s['name'].lower() == search.lower():
                    print(f"✅ Found: {s}")
                    found = True
            if not found:
                print("❌ Student not found.")
    
        elif choice == '4':
            print("📚 Available Courses:")
            for c in available_courses:
                print(f"- {c}")
    
        elif choice == '5':
            print("🏆 Unique Clubs:")
            for club in all_clubs:
                print(f"- {club}")
    
        elif choice == '6':
            print("👋 Exiting. Data saved.")
            break
    
        else:
            print("❌ Invalid choice.")

      import json

                  print(f"✅ {products[pid]['name']} added to cart.")
              else:
                  print("❌ Invalid Product ID.")
      
          elif choice == '3':
              if not cart:
                  print("🛒 Your cart is empty.")
              else:
                  print("\n🧾 Cart Items:")
                  total = 0
                  for pid in cart:
                      p = products[int(pid)]
                      print(f"- {p['name']} | ₹{p['price']} | Brand: {p['brand']}")
                      total += p['price']
                  print(f"💰 Total: ₹{total}")
      
          elif choice == '4':
              pid = input("Enter Product ID to remove: ")
              if pid in cart:
                  cart.remove(pid)
                  brands = {products[int(pid)]['brand'] for pid in cart}
                  print("❎ Item removed from cart.")
              else:
                  print("❌ Item not in cart.")
      
          elif choice == '5':
              print("\n🔁 Unique Brands in Cart:")
              for brand in brands:
                  print(f"- {brand}")
      
          elif choice == '6':
              with open(filename, "w") as file:
                  json.dump(cart, file)
              print("💾 Cart saved. Exiting. Goodbye! 👋")
              break
      
          else:
              print("❌ Invalid choice.")
