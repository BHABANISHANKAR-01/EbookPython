🎯 What are Control Flow Statements?

Control flow statements are like traffic signals for your code 🚦. They determine which path your program takes and how many times it repeats certain actions.

💡 Fun Fact: The term "control flow" was first introduced by computer scientist Maurice Wilkes in 1949!

Types of Control Flow:

  • Conditional Statements: Make decisions (if, else, elif)
  • Loops: Repeat actions (while, for)
  • Control Statements: Jump over code (break, continue, pass)

🤖 1. Conditional Statements – Making Decisions

Conditional statements let your program make decisions based on certain conditions, just like you decide what to wear based on the weather! 🌤️

Basic Syntax:

if condition:
    # run this block
elif another_condition:
    # run this if above fails
else:
    # fallback code
🧠 Real-Life Analogy:
Think of conditional statements like a smart thermostat:
• If temperature > 25°C → Turn on AC
• Else if temperature < 18°C → Turn on heater
• Else → Keep current setting

🎮 Practical Examples:

Example 1: Age Verification

age = 18
if age >= 18:
    print("You can vote! 🗳️")
else:
    print("Sorry, you are too young 😢")
Output:
You can vote! 🗳️

Example 2: Grade System

marks = 85
if marks >= 90:
    print("Grade: A+ 🏆")
elif marks >= 75:
    print("Grade: A 🎉")
elif marks >= 60:
    print("Grade: B 🙂")
else:
    print("Grade: Needs Improvement 😐")
Output:
Grade: A 🎉
⚠️ Important: Python uses indentation (spaces/tabs) to define code blocks. Always use consistent indentation!

🔁 2. Loops – Doing Things Repeatedly

Loops are like a broken record that keeps playing until you tell it to stop! 🎵

🌀 A. while Loop

The while loop repeats a block as long as a condition remains true.

count = 1
while count <= 5:
    print("Count:", count)
    count += 1
Output:
Count: 1
Count: 2
Count: 3
Count: 4
Count: 5
🧠 Real-Life Analogy:
Imagine eating Laddus until the box is empty:
• While box has Laddus → Keep eating
• When box is empty → Stop eating
Just like: while laddus_in_box: eat_laddu() 😋

🔂 B. for Loop

The for loop is used to iterate over sequences (lists, strings, ranges).

Basic Range Loop:

for i in range(1, 6):
    print("This is line", i)
Output:
This is line 1
This is line 2
This is line 3
This is line 4
This is line 5

Looping Through Lists:

fruits = ["apple", "banana", "mango"]
for fruit in fruits:
    print("I like", fruit)
Output:
I like apple
I like banana
I like mango
💡 Pro Tip: range(3) gives you [0, 1, 2], not [1, 2, 3]! Python starts counting from 0.

🛑 3. Loop Control Statements

These are like emergency buttons for your loops! 🚨

⛔ break – Exit the loop immediately

Like hitting the emergency stop button!

for i in range(1, 10):
    if i == 5:
        break
    print(i)
Output: 1 2 3 4

⏩ continue – Skip to next iteration

Like skipping a song you don't like!

for i in range(1, 6):
    if i == 3:
        continue
    print(i)
Output: 1 2 4 5

😶 pass – Do nothing (a placeholder)

Like saying "I'll do this later"

for i in range(3):
    pass  # do nothing for now
🧠 Memory Trick:
• break = "Break out of here!"
• continue = "Continue to the next one"
• pass = "Pass this for now"

🧪 Interactive Practice: Guess the Number Game 🎲

Let's create a simple number guessing game using control flow!

import random

secret = random.randint(1, 10)
attempts = 0
max_attempts = 3

print("🎮 Welcome to Guess the Number!")
print("I'm thinking of a number between 1 and 10.")

while attempts < max_attempts:
    guess = int(input("Enter your guess: "))
    attempts += 1
    
    if guess == secret:
        print("🎉 You guessed it right!")
        break
    elif guess < secret:
        print("Too low! Try again.")
    else:
        print("Too high! Try again.")

if attempts == max_attempts and guess != secret:
    print(f"Game Over! The number was {secret}")

🔐 Advanced Example: Mini Login System

Let's build a simple login system using control flow concepts!

# Predefined password
correct_password = "python123"

# Attempt counter
attempts = 0
max_attempts = 3

print("🔐 Welcome to the Login System")

while attempts < max_attempts:
    entered_password = input("Enter your password: ")
    
    if entered_password == correct_password:
        print("✅ Welcome! Access granted.")
        break  # Exit loop if password is correct
    else:
        attempts += 1
        remaining = max_attempts - attempts
        print(f"❌ Incorrect password. {remaining} attempts remaining.")

if attempts == max_attempts:
    print("🚫 Access Denied. Too many failed attempts.")
Sample Output:
🔐 Welcome to the Login System
Enter your password: wrong123
❌ Incorrect password. 2 attempts remaining.
Enter your password: python123
✅ Welcome! Access granted.
