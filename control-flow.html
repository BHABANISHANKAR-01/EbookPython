üéØ What are Control Flow Statements?

Control flow statements are like traffic signals for your code üö¶. They determine which path your program takes and how many times it repeats certain actions.

üí° Fun Fact: The term "control flow" was first introduced by computer scientist Maurice Wilkes in 1949!

Types of Control Flow:

  • Conditional Statements: Make decisions (if, else, elif)
  • Loops: Repeat actions (while, for)
  • Control Statements: Jump over code (break, continue, pass)

ü§ñ 1. Conditional Statements ‚Äì Making Decisions

Conditional statements let your program make decisions based on certain conditions, just like you decide what to wear based on the weather! üå§Ô∏è

Basic Syntax:

    # run this block
elif another_condition:
    # run this if above fails
else:
    # fallback code
üß† Real-Life Analogy:

‚Ä¢ If temperature > 25¬∞C ‚Üí Turn on AC
‚Ä¢ Else if temperature < 18¬∞C ‚Üí Turn on heater
‚Ä¢ Else ‚Üí Keep current setting

üîÅ 2. Loops ‚Äì Doing Things Repeatedly

Loops are like a broken record that keeps playing until you tell it to stop! üéµ

üåÄ A. while Loop

The while loop repeats a block as long as a condition remains true.

count = 1
while count <= 5:
    print("Count:", count)
    count += 1
Output:
Count: 1
Count: 2
Count: 3
Count: 4
Count: 5
üß† Real-Life Analogy:
Imagine eating Laddus until the box is empty:
‚Ä¢ While box has Laddus ‚Üí Keep eating
‚Ä¢ When box is empty ‚Üí Stop eating
Just like: while laddus_in_box: eat_laddu() üòã

üîÇ B. for Loop

The for loop is used to iterate over sequences (lists, strings, ranges).

Basic Range Loop:

for i in range(1, 6):
    print("This is line", i)
Output:
This is line 1
This is line 2
This is line 3
This is line 4
This is line 5

Looping Through Lists:

fruits = ["apple", "banana", "mango"]
for fruit in fruits:
    print("I like", fruit)
Output:
I like apple
I like banana
I like mango
üí° Pro Tip: range(3) gives you [0, 1, 2], not [1, 2, 3]! Python starts counting from 0.

üõë 3. Loop Control Statements

These are like emergency buttons for your loops! üö®

‚õî break ‚Äì Exit the loop immediately

Like hitting the emergency stop button!

for i in range(1, 10):
    if i == 5:
        break
    print(i)
Output: 1 2 3 4

‚è© continue ‚Äì Skip to next iteration

Like skipping a song you don't like!

for i in range(1, 6):
    if i == 3:
        continue
    print(i)
Output: 1 2 4 5

üò∂ pass ‚Äì Do nothing (a placeholder)

Like saying "I'll do this later"

for i in range(3):
    pass  # do nothing for now
üß† Memory Trick:
‚Ä¢ break = "Break out of here!"
‚Ä¢ continue = "Continue to the next one"
‚Ä¢ pass = "Pass this for now"

üß™ Interactive Practice: Guess the Number Game üé≤

Let's create a simple number guessing game using control flow!

import random

secret = random.randint(1, 10)
attempts = 0
max_attempts = 3

print("üéÆ Welcome to Guess the Number!")
print("I'm thinking of a number between 1 and 10.")

while attempts < max_attempts:
    guess = int(input("Enter your guess: "))
    attempts += 1
    
    if guess == secret:
        print("üéâ You guessed it right!")
        break
    elif guess < secret:
        print("Too low! Try again.")
    else:
        print("Too high! Try again.")

if attempts == max_attempts and guess != secret:
    print(f"Game Over! The number was {secret}")

üîê Advanced Example: Mini Login System

Let's build a simple login system using control flow concepts!

# Predefined password
correct_password = "python123"

# Attempt counter
attempts = 0
max_attempts = 3

print("üîê Welcome to the Login System")

while attempts < max_attempts:
    entered_password = input("Enter your password: ")
    
    if entered_password == correct_password:
        print("‚úÖ Welcome! Access granted.")
        break  # Exit loop if password is correct
    else:
        attempts += 1
        remaining = max_attempts - attempts
        print(f"‚ùå Incorrect password. {remaining} attempts remaining.")

if attempts == max_attempts:
    print("üö´ Access Denied. Too many failed attempts.")
