🛠️ Step-by-Step: How to Start with NumPy

✅ Step 1: Install NumPy (if not installed)

pip install numpy

✅ Step 2: Import NumPy

import numpy as np

We use np as a common alias to shorten code.

📦 1. Creating Arrays

➤ 1D Array

arr = np.array([1, 2, 3, 4])
  print(arr)  # Output: [1 2 3 4]

➤ 2D Array (Matrix)

matrix = np.array([[1, 2], [3, 4]])
  print(matrix)

📦 Output:

[[1 2]
  [3 4]]

⚙️ 2. Useful Array Operations

arr = np.array([1, 2, 3, 4])
  print(arr + 5)          # ➤ [6 7 8 9]
  print(arr * 2)          # ➤ [2 4 6 8]
  print(np.mean(arr))     # ➤ Average = 2.5
  print(np.sqrt(arr))     # ➤ Square roots

🧠 Explanation:

NumPy applies operations element-wise (called vectorized operations).

You don't need loops!

🧪 3. More Array Functions

print(np.max(arr))      # ➤ 4
  print(np.min(arr))      # ➤ 1
  print(np.sum(arr))      # ➤ 10
  print(np.std(arr))      # ➤ Standard deviation

📏 4. Array Shapes and Dimensions

print(arr.shape)        # (4,) → 1D with 4 elements
  print(matrix.shape)     # (2, 2) → 2x2 matrix
  print(arr.ndim)         # ➤ 1 (1D array)
  print(matrix.ndim)      # ➤ 2 (2D array)

🔁 5. Creating Special Arrays

np.zeros((2, 3))     # 2x3 array of zeros
  np.ones((3, 2))      # 3x2 array of ones
  np.arange(0, 10, 2)  # [0 2 4 6 8]
  np.linspace(1, 5, 4) # 4 numbers from 1 to 5

🔄 6. Reshaping Arrays

arr = np.array([1, 2, 3, 4, 5, 6])
  reshaped = arr.reshape((2, 3))
  print(reshaped)

Output:

[[1 2 3]
  [4 5 6]]

🔎 7. Indexing & Slicing

arr = np.array([10, 20, 30, 40])
  print(arr[1])       # ➤ 20
  print(arr[1:3])     # ➤ [20 30]

2D Example:

matrix = np.array([[5, 6], [7, 8]])
  print(matrix[1][0])  # ➤ 7

🎮 Challenge Example

import numpy as np
  
  marks = np.array([80, 90, 70, 85])

📥 1. Importing pandas & Creating DataFrames

import pandas as pd
  
  data = {
      "Name": ["Aman", "Elon", "John"],
      "Marks": [90, 85, 88]
  }
  
  df = pd.DataFrame(data)
  print(df)

📦 Output:

   Name  Marks
  0  Aman     90
  1  Elon     85
  2  John     88

🔎 DataFrame is like a table (rows & columns), and each column is a Series (like a list with labels).

🔍 2. Accessing Data in DataFrame

📌 Access a Column (like Excel header)

print(df["Name"])

Output:

0    Aman
  1    Elon
  2    John
  Name: Name, dtype: object

📌 Access a Row by Label (using .loc[])

print(df.loc[0])

Output:

Name     Aman
  Marks      90
  Name: 0, dtype: object

📌 Access by Index Position (using .iloc[])

print(df.iloc[1])

Output:

Name     Elon
  Marks      85
  Name: 1, dtype: object

📈 3. Descriptive Statistics

🔢 Basic Summary

print(df.describe())

Output:

          Marks
  count   3.000000
  mean   87.666667
  std     2.516611
  min    85.000000
  25%    86.500000
  50%    88.000000
  75%    89.000000
  max    90.000000

📊 Mean of Marks

print(df.mean(numeric_only=True))

Output:

Marks    87.666667
  dtype: float64

📤 4. Reading from a CSV File

Suppose you have a file called students.csv:

df = pd.read_csv("students.csv")
  print(df.head())

read_csv() → loads data from a CSV file

head() → shows first 5 rows (default)

You can also use:

df.tail()      # Last 5 rows
  df.columns     # Column names
  df.shape       # (rows, columns)
